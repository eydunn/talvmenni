Abstract

The goal of the Talvmenni project is to develop an open source distributed chess engine based on Java (and also on Jini/JavaSpaces). Most chess-engines are written in c/c++ and on what seems to be a rather machine-near abstraction level. The first goal of the project is to explore the possibilty of using a higher abstraction level and still using efficient data structures like bit-boards. The second goal of the project is to see if using the master/worker idiom in JavaSpaces can be utilized in this context. The sourcecode will be written in Java (J2SE 5.0) and thus will run on any OS that supports Java (J2SE 5.0).
