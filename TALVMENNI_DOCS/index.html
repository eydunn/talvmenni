Abstract

The goal of the Talvmenni project is to develop an OpenSource chess engine written in Java. We want to demonstrate that Java can be tuned to a point, where it is almost up to the performance of an application, that was directly compiled to native code. Even clean object-oriented design does not necessarily generate an overhead, that makes high-performance computing virtually impossible.
